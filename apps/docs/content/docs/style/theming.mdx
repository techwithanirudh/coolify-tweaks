Coolify Tweaks supports custom themes from [TweakCN](https://tweakcn.com), a theme editor for shadcn/ui components. You can apply any theme from the TweakCN gallery or create your own.

## Getting a theme ID

1. Visit [TweakCN Theme Editor](https://tweakcn.com/editor/theme)
2. Browse themes or customize your own
3. Click the share button and copy the theme ID. If it's a default theme, it contains its name like `claude`, or if it's a custom theme, it starts with `c` followed by 24 characters (e.g., `cmh4ecxjc000404l78qwda7o0`)

## Applying themes

<ModeContent>
  <ModeContentStylus>
    ### Stylus installation

    #### To apply a theme:

    Simply open the URL with your theme ID in a new tab. This will automatically overwrite the existing theme:

    <ThemeCode
      lang="text"
      code="https://coolify-tweaks-api.techwithanirudh.com/release/latest/?theme=YOUR_THEME_ID&asset=main.user.css"
    />

    > **Note:** The theme query parameter should come first, as Stylus identifies user styles by URLs ending with `user.css`

    #### To change themes:

    Open the new theme URL in a new tab to overwrite the current theme.

    #### To remove a theme:

    1. Open Stylus extension settings
    2. Find "coolify-tweaks" in your styles list
    3. Click the **X** icon to remove the theme

  </ModeContentStylus>
  <ModeContentDynamic>
    ### Dynamic Config installation

    #### To apply a theme:

    1. Edit the dynamic config file:
       ```
       /data/coolify/proxy/dynamic/coolify-tweaks.yml
       ```

    2. Find the `rb-inject-css` middleware section and update the `replacement` URL to include your theme ID:

    <ThemeCode
      lang="yaml"
      code={`rb-inject-css:
  plugin:
    rewritebody:
      lastModified: true
      rewrites:
        - regex: "(?is)</head>"
          replacement: '<link rel="stylesheet" href="https://coolify-tweaks-api.techwithanirudh.com/release/latest/?theme=YOUR_THEME_ID&asset=main.css" referrerpolicy="no-referrer" crossorigin="anonymous"></head>'`}
    />

    3. **Restart Traefik:**

       Go to **Servers → localhost → Proxy → Restart Proxy**

       ![Restart Proxy](/assets/installation/dynamic-config/restart-proxy.png)

    4. **Refresh your browser** to see the new theme

    #### To change themes:

    - Edit the YAML file and replace `YOUR_THEME_ID` with a new theme ID
    - Restart Traefik and refresh your browser

    #### To remove a theme:

    - Edit the YAML file and remove `?theme=YOUR_THEME_ID&` from the URL:
      ```yaml
      replacement: '<link rel="stylesheet" href="https://coolify-tweaks-api.techwithanirudh.com/release/latest/?asset=main.css" referrerpolicy="no-referrer" crossorigin="anonymous"></head>'
      ```
    - Restart Traefik and refresh your browser

  </ModeContentDynamic>
</ModeContent>

## Custom Themes

If you have your own theme CSS and want to make it work with Coolify Tweaks, here's how you can do it:

### 1. Create a new CSS file

First, find the CSS you want to use and copy both the light and dark mode contents. (These should follow the ShadCN theme spec.)

```css
:root {
  --background: oklch(1 0 0);
  --foreground: oklch(0.145 0 0);

  --primary: oklch(0.205 0 0);
  /* ... */
}

.dark {
  --background: oklch(0.233 0.014 304.7);
  --foreground: oklch(0.9938 0.0013 106.4231);

  --primary: oklch(0.922 0 0);
  /* ... */
}
```

### 2. Import the Theme into TweakCN

Go to [TweakCN Theme Editor](https://tweakcn.com/editor/theme) and click the "Import" button.

Then, paste the CSS file contents into the "CSS" field and click the "Import" button.

![Import Theme](/assets/installation/themes/import-theme.png)

### 3. Share the Theme

Click the "Share" button and authenticate with TweakCN.

![Share Theme](/assets/installation/themes/tweakcn-auth.png)

After authenticating, you will be asked to name your theme.

![Name Theme](/assets/installation/themes/name-theme.png)

Now, you'll see the theme ID in the URL. Copy it.

![Theme ID](/assets/installation/themes/share-theme.png)

> Note: The theme ID is the string after `tweakcn.com/themes/` in the URL. It will look something like `cmh4ecxjc000404l78qwda7o0`.

### 4. Apply the Theme

Now, you can apply the theme to your Coolify Tweaks instance.

```
https://coolify-tweaks-api.techwithanirudh.com/release/latest/?theme=THEME_ID&asset=main.user.css
```

Replace `THEME_ID` with the theme ID you copied.

That's all! You can now use your custom theme with Coolify Tweaks.

## Tips

- **Theme updates:** When you update a theme in TweakCN, the changes apply automatically since the theme ID stays the same
- **Browser cache:** If theme changes don't appear, try a hard refresh (`Ctrl+Shift+R` or `Cmd+Shift+R`)
