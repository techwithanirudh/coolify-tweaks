# Contributing

Thanks for thinking about contributing to this project!

## Guide

You can either use the Dev Container in this repository, or have a system with [Bun](https://bun.com) installed.

1. Clone your fork of this repository with `git clone`.

2. Install dependencies:

   ```bash
   bun install
   ```

3. Start the development server:

   ```bash
   bun run dev
   ```

4. Open <http://localhost:3000> in your browser with Stylus installed. Stylus should automatically prompt you to install the userstyle. **Make sure "Live Preview" is on for automatic reloading in development!**

5. Make your changes to the code.

6. Create a changeset for your changes:

   ```bash
   bun run changeset
   ```

7. Commit your changes with Git.

8. Create a Pull Request to this repository!

Yay, you have finished the contribution process! Hopefully your PR will be merged!
